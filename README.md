# Backend Intro API

Proyek backend sederhana menggunakan Node.js dan Express.js dengan struktur modular.

## Fitur
- Modular route structure.
- RESTful API endpoints.
- Request logging middleware.

## Instalasi
1. git clone https://github.com/kar1mmmm/backend-intro.git
2. cd backend-intro
3. npm install
4. node index.js

## Endpoint
- GET / : Home
- GET /api/sekolah/:jenjang : Mengambil data sekolah (SMA/SMP/SD)
